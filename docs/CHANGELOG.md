## [0.1.0] - 2026-02-04

### Added
- 12-page responsive website built with Next.js 15, React 19, TypeScript 5, and Tailwind CSS 4
- Home page with Three.js WebGL shader animation hero, service overview, and CTAs
- About page with CEO bio, company values, timeline, and certifications
- Services page with interactive explorer, pricing toggle (monthly/annual), and confetti effects
- Travel page with Mapbox GL interactive map (30+ destinations), 4-step trip planner wizard, and regional destination tabs
- Contact page with contact form and booking calendar
- FAQ page with category-filtered accordion
- Legal pages: Privacy Policy, Terms of Service, Cookie Policy, Copyright Policy
- HTML sitemap page
- AI chatbot interface with guided conversation flows
- Exit-intent popup with GoHighLevel CRM webhook integration
- Cookie consent banner with preference management and CRM tracking
- Responsive navbar with mobile hamburger menu
- 4-column footer with social links
- 7 downloadable PDF white papers
- Full SEO optimization: per-page metadata, Open Graph, Twitter Cards, JSON-LD structured data
- Google Analytics 4 integration
- Animated gradient button component with CSS custom property transitions
- shadcn/ui component system (Button, Label, Switch) with Radix UI primitives
- Design system with CSS custom properties: primary (#161821), accent (#c1c5b3), muted (#f0f1ec)
- Inter and Playfair Display fonts via next/font
- Complete project documentation: README, CLAUDE.md, SETUP, ARCHITECTURE, DEPLOYMENT, CONTRIBUTING, CHANGELOG, LICENSE, .env.example
